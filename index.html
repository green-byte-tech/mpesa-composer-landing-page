How to Use MPESAComposer

1. Start the Bot

Search for @MPESAComposerBot on Telegram and click "Start"

Available Commands:

  • /start - Begin using the bot
  • /help - View all available commands
  • /setgroup - Set default group name
  • /template - Choose message format

2. Forward Messages

Forward your M-PESA messages directly to the bot. The bot accepts:

  • Single M-PESA messages
  • Multiple messages at once
  • Messages from different dates
