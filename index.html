What is MPESAComposer?

MPESAComposer is a powerful Telegram bot that simplifies the process of managing M-PESA contributions for WhatsApp groups. It automatically parses M-PESA messages and converts them into well-formatted contribution reports that can be easily shared on WhatsApp.

Key Features

  • Instant M-PESA message parsing
  • Smart transaction data extraction
  • Customizable report templates
  • WhatsApp-friendly formatting
  • Real-time processing
  • User-friendly Telegram interface

How It Works

1

Forward M-PESA Message

Simply forward your M-PESA confirmation message to our Telegram bot

2

Automatic Processing

The bot extracts key information like amount, sender, date, and transaction ID

3

Get Formatted Report

Receive a beautifully formatted report ready to share on WhatsApp

Getting Started

  1. Open Telegram and search for @MPESAComposerBot
  2. Start the bot with the /start command
  3. Choose your preferred template using /templates
  4. Forward any M-PESA message to get your formatted report
